Undo

Allows you to undo almost all changes made since the file was last loaded. Some changes cannot be undone. When such changes are made, you will be prompted with a pop-up dialog. Undo is not available when there are no changes that can be undone. Both the Undo menu item and button will be grayed out when it is not an allowed function.
