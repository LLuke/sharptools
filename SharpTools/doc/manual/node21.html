Set Password...

Keyboard Shortcut: None.
Allows a password to be set for the current spreadsheet; begins Password dialog.



Enter your desired password in the first field, then type it exactly again in the second field. If the passwords do not match, you will be prompted to enter the password again. If you wish to open the file again in the future, you will have to enter the password.
