Find...



Find allows for two different types of searching: case sensitive and case insensitive. This is not the normal upper-case or lower-case distinction of case, but rather whether you want to match the search string to any cell with it, or only cells with the full string. For example, the string test will match Test 1, Test 2, Tests, Test Grade, and Test 1 if Match Case is selected. If Match Entire Case Only is selected, only Test will be matched. The default behavior is to match the case. Find will not return any value contained within a formula string. If no string is found that matches the search string, a pop-up error will be returned.
