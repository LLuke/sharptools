Toolbar

  • New: Create new file.
  • Open: Load a file.
  • Save: Save the current file; grayed out when nothing to be saved.
  • Set Password: Set a password on the current file; image changes to a lock when file already has a password set.
  • Print: Prints the current file.
  • Undo: Undo change back to last load; grayed out when nothing to undo.
  • Redo: Redo to last change; grayed out when nothing to redo.
  • Cut: Cut selected range.
  • Copy: Copy selected range.
  • Paste: Paste clipboard contents into spreadsheet.
  • Find: Search for an entry in the spreadsheet.
  • Insert Row: Insert row or rows into spreadsheet.
  • Insert Column: Insert column or columns into spreadsheet.
  • Delete Row: Remove row or rows from spreadsheet.
  • Delete Row: Remove column or columns from spreadsheet.
  • Sort: Sort selected columns.
  • Histogram: Generate histogram from selected range.
  • Help: Open help browser.
  • Function Help: Select an entry from the drop-down to get help on the function; when a cell is editable, select an entry to insert the function in the cell.
